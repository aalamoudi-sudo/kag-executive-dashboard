# سجل إضافة مركز الإشعارات — KAG Cinematic Strategic Platform

## الهدف
إضافة مركز إشعارات داخل المنصة يتيح إرسال إشعارات فردية أو جماعية لمديري المسارات، مع تحديد مستوى الأهمية وموعد التسليم أو التحديث المطلوب.

## ما تمت إضافته
- صفحة جديدة في القائمة الجانبية باسم: **مركز الإشعارات**.
- نموذج إنشاء إشعار يدعم:
  - إرسال فردي لمدير مسار واحد.
  - إرسال جماعي لأكثر من مدير مسار.
  - إرسال لجميع مديري المسارات.
- حقول تشغيلية للإشعار:
  - العنوان.
  - نص المطلوب.
  - مستوى الأهمية: حرجة / عالية / متوسطة / منخفضة.
  - نوع الإجراء: تسليم / تحديث / قرار / متابعة.
  - موعد التسليم.
  - موعد التحديث المطلوب.
- لوحة متابعة للإشعارات تعرض:
  - إجمالي الإشعارات.
  - الإشعارات الحرجة.
  - الإشعارات العالية.
  - الإشعارات المرسلة.
  - الإشعارات المرتبطة بموعد تسليم.
- سجل إشعارات مرئي داخل المنصة مع المستلمين والأهمية والمواعيد والحالة.

## التعديلات التقنية
- تعديل `public/index.html` لإضافة صفحة مركز الإشعارات وزر التنقل.
- تعديل `public/script.js` لإضافة منطق إنشاء الإشعارات، قراءة المستلمين، إرسال البيانات، وعرض السجل.
- تعديل `public/style.css` لإضافة تصميم سينمائي خاص بمركز الإشعارات.
- تعديل `server.js` لإضافة API جديد:
  - `GET /api/notifications` لقراءة سجل الإشعارات.
  - `POST /api/notifications` لإرسال إشعار جديد.
- إضافة تخزين محلي للإشعارات في ملف `notifications_store.json` عند التشغيل.
- دعم Webhook اختياري عبر متغير البيئة `NOTIFICATION_WEBHOOK_URL` للتكامل لاحقًا مع البريد أو Teams أو WhatsApp Business أو أي قناة خارجية.

## ضوابط الصلاحيات
- قراءة الإشعارات متاحة للمستخدمين المسجلين.
- إرسال الإشعارات يتطلب صلاحية Admin.
- جميع طلبات الإرسال تمر عبر فحص Same-Origin.
- يتم تنظيف النصوص المدخلة قبل التخزين.

## متغيرات بيئة اختيارية
```env
NOTIFICATION_WEBHOOK_URL=https://example.com/webhook
NOTIFICATION_STORE=notifications_store.json
```

## مقترحات التطوير القادمة
- إضافة تأكيد استلام من مدير المسار.
- إضافة حالة: مقروء / غير مقروء / تم التنفيذ / متأخر.
- إضافة تذكير تلقائي قبل موعد التسليم.
- إضافة تصعيد تلقائي عند تجاوز الموعد.
- ربط الإشعار مباشرة بالمؤشر أو الخطر أو القرار المرتبط به.
- إضافة قنوات فعلية مثل البريد وTeams وWhatsApp Business API.
